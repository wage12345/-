# Codex 辅助开发记录

## 1. 项目结构设计

提示词：

请为基于 Express + EJS + MySQL 的学生信息管理系统设计目录结构。

用途：

用于确定项目中的 routes、controllers、models、views、config 等目录。

## 2. 数据库连接

提示词：

请为 Express + mysql2 项目生成一个 config/db.js，使用 .env 中的数据库配置创建连接池，并导出 pool。

用途：

用于生成 MySQL 连接池代码。

## 3. 登录模块

提示词：

请为学生信息管理系统生成登录模块，包括路由、控制器、用户查询和 session 鉴权逻辑。

用途：

用于实现管理员登录和登录状态保持。

## 4. 学生 CRUD

提示词：

请为 students 表生成增删改查功能，包括 model、controller、routes 和 EJS 页面。

用途：

用于实现学生信息管理核心功能。

## 5. 报错修复

提示词：

请根据以下 Express 报错信息分析原因并给出修改建议。
